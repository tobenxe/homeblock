
# Homeblock
A customizable dashboard for your new tab.

![Homeblock image](./chrome/icons/icon128.png)

Homeblock is made with React.js, Webpack, Babel & Tested with Jest

## Features 
- Add custom background page
- Pick blocks to display
Blocks Available:
- Current Date + Time.
- Weather
- Todo
