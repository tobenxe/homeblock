
# Homeblocks
A customizable dashboard for your new tab.

![Homeblocks image](./chrome/icons/icon128.png)

## Features 
- Add custom background page
- Pick blocks to display
Blocks Available:
- Current Date + Clock.
- Weather
- Todo
